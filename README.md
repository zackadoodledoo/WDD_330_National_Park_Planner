# Adventure Planner (WDD 330)

A single-page application that helps users discover National Parks, view park details and weather, plan trips, and manage packing lists.

## Features
- Search National Parks (NPS API)
- View park details and 5-day weather forecast
- Create and save trip plans
- Manage packing checklists
- Persistent storage using localStorage

## Tech Stack
- HTML
- CSS
- JavaScript (ES Modules)
- National Park Service API
- OpenWeather API

## Setup
1. Clone the repo
2. Run a local server (Live Server or similar)
3. Add API keys to `config.js`
4. Create `src/config.js` using `config.example.js` and add your API keys.
